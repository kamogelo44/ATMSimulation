# ATM Simulation Project

A simple Java program that works like a real ATM machine.

## What This Program Does

- Asks for your User ID and PIN to login
- Lets you check your account balance
- Allows you to deposit money
- Allows you to withdraw money
- Keeps working until you choose to exit

## How to Run

1. Make sure you have Java installed
2. Open the project in your code editor
3. Run the `ATMSimulation.java` file
4. Follow the instructions on screen

## Login Details (for testing)

- User ID: Any number (like 12345)
- PIN: 3252
